# AkademiTrack V1

**Automatisk frammøteregistrering for Akademiet-studenter**

## Hva er AkademiTrack?

AkademiTrack er en desktop-applikasjon designet spesielt for studenter ved Akademiet for automatisk å registrere frammøte for STU (studietid) timer. I stedet for å manuelt klikke "registrer" for hver studieperiode, håndterer dette programmet registreringen automatisk når registreringsvinduene åpner.

## Hvorfor bruke AkademiTrack?

- **Gå aldri glipp av registrering**: Registrerer deg automatisk for STU-perioder i løpet av deres utpekte tidsvindu
- **Sparer tid**: Ingen behov for å manuelt registrere for hver studieperiode
- **Ro i sjelen**: Ikke bekymre deg for å glemme å registrere for studietid
- **Enkel å bruke**: Enkelt grensesnitt med tydelige statusindikatorer

## Funksjoner

### 🚀 Hovedfunksjoner
- **Automatisk registrering**: Overvåker timeplanen din og registrerer frammøte for STU-timer automatisk
- **Manuell registrering**: Mulighet til å manuelt registrere for perioder ved behov
- **Oppsett og innlogging**: Enkelt engangsoppsett som lagrer påloggingsøkten din
- **Statusovervåking**: Tydelige indikatorer som viser når programmet kjører og hva som skjer

### 📱 Brukergrensesnitt
- Rent, moderne design med norsk grensesnitt
- Sanntidsstatusoppdateringer
- Innstillingsvindu med konsollutgang for overvåking
- Fremdriftsindikatorer under oppsett

### 🛡️ Smarte funksjoner
- Registrerer kun for STU (studietid) timer
- Respekterer registreringstidsvindu
- Stopper automatisk når alle perioder er fullført
- Øktadministrasjon med vedvarende cookies
- Feilhåndtering og gjenoppretting

## Hvordan det fungerer

1. **Oppsett**: Kjør "Oppsett og innlogging" for å autentisere med iSkole
2. **Start**: Klikk "Start automatisering" for å begynne automatisk overvåking
3. **Overvåk**: Programmet sjekker hvert minutt for registreringsmuligheter
4. **Registrer**: Registrerer automatisk frammøte i løpet av gyldige tidsvindu
5. **Fullfør**: Stopper automatisk når alle STU-perioder er registrert

## Systemkrav

- Windows, macOS eller Linux
- Internettilkobling
- Tilgang til iSkole (Akademiet studentkonto)
- Python 3.7+ (installerer automatisk nødvendige pakker)

## Installasjon

1. Last ned alle programfiler
2. Kjør hovedprogramfilen
3. Programmet vil automatisk installere manglende avhengigheter
4. Følg oppsettsveiviseren for å koble til iSkole-kontoen din

## Bruk

### Første gangs oppsett
1. Klikk **"Oppsett og innlogging"**
2. Fullfør pålogging i nettleservinduet som åpnes
3. Vent på "Oppsett fullført" bekreftelse

### Daglig bruk
1. Klikk **"Start automatisering"**
2. Programmet kjører i bakgrunnen
3. Sjekk statusindikator for gjeldende tilstand
4. Bruk **"Innstillinger"** for å se detaljerte logger

### Manuell registrering
- Bruk **"Manuell registrering"** hvis du trenger å registrere utenfor normal timeplan
- Nyttig for innhentingstimer eller spesielle situasjoner

## Statusindikatorer

- **🟢 Kjører**: Programmet overvåker aktivt og vil registrere når det er aktuelt
- **🔴 Feil**: Noe gikk galt - sjekk innstillingskonsollen for detaljer
- **🟡 Oppsett nødvendig**: Påloggingsøkt utløpt, kjør oppsett igjen
- **⚪ Klar**: Programmet er klart til å starte

## Viktige merknader

- Fungerer kun for STU (studietid) timer
- Respekterer de offisielle registreringstidsvinduene
- Forstyrrer ikke vanlige akademiske timer
- Designet spesielt for Akademiets iSkole-system

## Feilsøking

**Programmet starter ikke automatisering:**
- Kjør "Oppsett og innlogging" først
- Sjekk internettilkobling

**Registrering mislykkes:**
- Økten din kan ha utløpt
- Kjør oppsett igjen for å oppdatere pålogging

**Nettleservinduet lukker seg ikke:**
- Fullfør påloggingsprosessen helt
- Naviger til timeplan-siden hvis omdirigert andre steder

## Sikkerhet og personvern

- Påloggingsinformasjonen din lagres aldri
- Kun økt-cookies lagres lokalt
- Programmet får kun tilgang til dine egne frammøtedata
- Ingen data deles med tredjeparter

# AkademiTrack V1

**Automatic attendance registration for Akademiet students**

## What is AkademiTrack?

AkademiTrack is a desktop application designed specifically for students at Akademiet to automatically register attendance for STU (studietid) classes. Instead of manually clicking "register" for each study period, this program handles the registration automatically when the registration windows open.

## Why use AkademiTrack?

- **Never miss registration**: Automatically registers you for STU periods during their designated time windows
- **Saves time**: No need to manually register for each study period
- **Peace of mind**: Don't worry about forgetting to register for studietid
- **Easy to use**: Simple interface with clear status indicators

## Features

### 🚀 Main Functions
- **Automatic Registration**: Monitors your schedule and registers attendance for STU classes automatically
- **Manual Registration**: Option to manually register for periods when needed
- **Setup & Login**: Easy one-time setup that saves your login session
- **Status Monitoring**: Clear indicators showing when the program is running and what's happening

### 📱 User Interface
- Clean, modern design with Norwegian interface
- Real-time status updates
- Settings window with console output for monitoring
- Progress indicators during setup

### 🛡️ Smart Features
- Only registers for STU (study time) classes
- Respects registration time windows
- Automatically stops when all periods are completed
- Session management with cookie persistence
- Error handling and recovery

## How it works

1. **Setup**: Run "Oppsett og innlogging" to authenticate with iSkole
2. **Start**: Click "Start automatisering" to begin automatic monitoring
3. **Monitor**: The program checks every minute for registration opportunities
4. **Register**: Automatically registers attendance during valid time windows
5. **Complete**: Stops automatically when all STU periods are registered

## Requirements

- Windows, macOS, or Linux
- Internet connection
- Access to iSkole (Akademiet student account)
- Python 3.7+ (automatically installs required packages)

## Installation

1. Download all program files
2. Run the main program file
3. The program will automatically install any missing dependencies
4. Follow the setup wizard to connect your iSkole account

## Usage

### First Time Setup
1. Click **"Oppsett og innlogging"**
2. Complete login in the browser window that opens
3. Wait for "Setup completed" confirmation

### Daily Use
1. Click **"Start automatisering"** 
2. The program runs in background
3. Check status indicator for current state
4. Use **"Innstillinger"** to view detailed logs

### Manual Registration
- Use **"Manuell Registrering"** if you need to register outside normal schedule
- Useful for makeup classes or special situations

## Status Indicators

- **🟢 Running**: Program is actively monitoring and will register when appropriate
- **🔴 Error**: Something went wrong - check settings console for details
- **🟡 Setup needed**: Login session expired, run setup again
- **⚪ Ready**: Program is ready to start

## Important Notes

- Only works for STU (studietid) classes
- Respects the official registration time windows
- Does not interfere with regular academic classes
- Designed specifically for Akademiet's iSkole system

## Troubleshooting

**Program won't start automation:**
- Run "Oppsett og innlogging" first
- Check internet connection

**Registration failing:**
- Your session may have expired
- Run setup again to refresh login

**Browser window doesn't close:**
- Complete the login process fully
- Navigate to the timeplan page if redirected elsewhere

## Safety & Privacy

- Your login credentials are never stored
- Only session cookies are saved locally
- Program only accesses your own attendance data
- No data is shared with third parties
